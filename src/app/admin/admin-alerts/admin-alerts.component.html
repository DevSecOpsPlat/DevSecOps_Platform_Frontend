Alertes

Incidents hors norme — attaques, verrouillages, changements d'identifiants, accès refusés.

Uniquement dans les alertes

  • Connexion échouée — chaque tentative incorrecte (avec IP et compteur)
  • Force brute — 3 échecs consécutifs → verrouillage 15 min
  • Takeover — mot de passe ou e-mail modifié par l'utilisateur
  • Accès refusé — tentative d'accès à l'API admin (HTTP 403)

Les connexions réussies, créations de compte, e-mails d'activation et actions admin courantes sont dans le Journal d'audit.

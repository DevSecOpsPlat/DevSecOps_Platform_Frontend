# Images requises par l'application

Placez dans ce dossier les fichiers suivants pour que les logos et icônes s'affichent :

| Fichier | Utilisé dans |
|---------|--------------|
| `attijari.png` | Navbar – logo Attijari Bank |
| `enviroTestLogorembg.png` | Navbar – logo EnviroTest |
| `user.png` | Navbar, Profil – icône utilisateur |
| `Logo.png` | Home – illustration hero |
| `Logormbg.png` | Sign-in, Sign-up – logo (sans fond) |
| `security.png` | Home – icône sécurité |
| `git.png` | Home – icône Git |
| `devsecops.png` | Home – icône pipeline |
| `gitIcon.png` | Environment Create – icône dépôt |
| `brancheIcon.png` | Environment Create – icône branche |
| `timerIcon.png` | Environment Create – icône timeout |

Noms exacts (respecter la casse). Formats supportés : PNG, JPG, SVG.
